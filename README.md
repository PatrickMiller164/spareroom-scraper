# SpareRoom Scraper

A Python tool for scraping rental listings from SpareRoom.co.uk, maintaining a local database of previously-seen rooms, filtering out expired listings, and exporting final results to Excel (and generating a map).

This script automates:
	•	Searching SpareRoom listings within rent limits
	•	Identifying and processing new rooms
	•	Filtering out expired listings
	•	Scoring and ranking rooms
	•	Exporting the results to an Excel file
	•	Saving and updating a local .pkl database
	•	Creating a map of all valid rooms

# Usage

```
python main.py
```
